📖 프로젝트 소개
"Github 업로드 딸깍!" 은 로컬 컴퓨터의 특정 폴더를 깃허브 저장소와 실시간으로 동기화하여 파일 관리를 자동화해주는 프로그램입니다.

코딩에 집중하기도 바쁜데 Git 명령어까지 신경 쓰기 힘드셨죠? 이 프로그램을 사용하면 파일을 저장하는 순간 자동으로 깃허브에 업로드됩니다. 초보자도 쉽게 본인의 GitHub에 자료를 정리하고, 매일매일 잔디를 심는 즐거움을 느낄 수 있습니다.

🚀 주요 기능 하이라이트
1️⃣ 실시간 양방향 동기화 (Real-time Synchronization)
더 이상 수동으로 add, commit, push를 할 필요가 없습니다.

🔄 자동 업로드: 지정된 로컬 폴더에 파일이 추가되거나 수정되면, 프로그램이 즉시 감지하여 자동으로 깃허브에 업로드합니다.

🛡️ 안전한 삭제 (휴지통 기능): 로컬에서 파일을 삭제해도 깃허브에서는 즉시 사라지지 않습니다. 원격 저장소의 _recycle_bin 폴더로 안전하게 이동되어 실수로 인한 파일 유실을 방지합니다.

🧹 스마트 초기 동기화: 프로그램 시작 시 로컬과 원격 저장소의 상태를 비교합니다. 로컬에 없는 파일은 휴지통으로 보내고, 로컬에만 있는 파일은 업로드하여 완벽한 동기화 상태를 맞춥니다.

2️⃣ 사용자 친화적인 GUI (User-Friendly GUI)
직관적인 인터페이스로 누구나 쉽게 제어할 수 있습니다.

[여기에 메인 실행 화면 이미지를 넣어주세요. 예: main_interface.png]

직관적인 제어판과 실시간 로그 뷰어

🎛️ 간편한 제어: '동기화 & 업로드 시작', '업로드 종료' 버튼 하나로 프로세스를 제어하세요.

📝 실시간 로그: 파일이 처리되는 모든 과정을 로그 창을 통해 투명하게 확인할 수 있습니다.

🎨 다양한 테마: ttkbootstrap을 활용하여 취향에 맞는 다양한 UI 테마를 선택할 수 있습니다.

3️⃣ 강력한 설정 관리 (Settings Management)
복잡한 환경 설정도 GUI에서 간편하게 해결합니다.

[여기에 설정 창 이미지를 넣어주세요. 예: settings_window.png]

토큰 및 저장소 상세 설정 화면

깃허브 개인 액세스 토큰(PAT), 사용자 이름, 저장소명, 브랜치, 감시할 폴더 경로 등 모든 설정을 UI에서 관리합니다.

설정값은 config.json에 영구 저장되어 매번 입력할 필요가 없습니다. (토큰은 안전하게 별도 처리됩니다.)

🧩 보너스 플러그인: 백준 문제 찾기
코딩 공부를 위한 특별한 선물! 🎁 solved.ac와 연동하여 Class별 백준 문제 목록을 크롤링해 보여줍니다.

[여기에 플러그인 창 이미지를 넣어주세요. 예: baekjoon_plugin.png]

통합된 백준 문제 찾기 플러그인

별도의 창에서 원하는 난이도의 문제를 쉽게 찾아보세요.
